# 筆 PostCraft — X投稿ジェネレーター

Claude API を使って、ペルソナ別のX投稿を自動バッチ生成するローカルツール。

**クローンするだけで、けんた・ともこのキャラクターでポスト生成がすぐ始められます。**

## セットアップ

```bash
git clone https://github.com/emaru0724-spec/x-post-generator.git
cd x-post-generator

# 1. 依存パッケージのインストール
npm install

# 2. APIキーの設定
cp .env.example .env
# .env に ANTHROPIC_API_KEY を設定

# 3. 起動
npm start
# → http://localhost:3462 をブラウザで開く
```

## 同梱ペルソナ（すぐ使える）

| ペルソナ | 属性 | 月収 | 発信テーマ |
|---------|------|------|-----------|
| **ともこ** | 42歳パート主婦、夫+息子2人 | パート8万+古着副業8〜10万 | 家計改善・メルカリノウハウ・子育て×副業 |
| **けんた** | 30代会社員、残業月40h | 手取り25万+古着副業12万 | 残業×副業時間術・仕入れ実践テクニック |

アプリ起動後、サイドバーの「古着副業アカウント」を選択 → ペルソナを選ぶ → 生成開始。

## 機能

- **バッチ生成**: 7日分×3型(朝/昼/夜) = 21本を一括生成
- **自動採点**: 5項目(フック/共感/ボイス/具体性/CTA)で採点、S/A/B/Cランク判定
- **ペルソナ管理**: キャラクターの新規作成・編集が可能
- **ストック管理**: 生成した投稿をMarkdownファイルで保存・閲覧
- **カレンダー表示**: 7日間の投稿スケジュールを視覚化

## リポジトリ内のデータ構成

```
data/
├── ガイド/
│   ├── ブランドボイス.md    ← 口調ルール（ペルソナ別の差分含む）
│   └── ターゲット.md       ← ターゲットDNA（F1主婦層 + M1会社員層）
├── テンプレート/
│   └── x-post.md          ← 投稿テンプレート（型A朝/型B昼/型C夜）
├── 品質基準/
│   └── 採点ルブリック.md   ← 5軸採点基準（S/A/B/C/Dランク）
├── ペルソナ/
│   ├── けんた/
│   │   └── _persona.md    ← けんたのキャラ設定
│   └── ともこ/
│       └── _persona.md    ← ともこのキャラ設定
└── ストック/               ← 生成した投稿の保存先
```

## ペルソナのカスタマイズ

`data/ペルソナ/{名前}/_persona.md` を編集するか、アプリUI上で「+ 新規ペルソナ」から作成できます。

### ペルソナ定義の書き方

```markdown
# ペルソナ定義: 名前

## 基本属性
- 年齢: XX歳
- 職業: ○○
- 家族構成: ○○

## 収入・副業
- 本業: 月XX万円
- 副業: 月XX万円

## 性格・口調
- 一人称: 「僕」or「わたし」
- 語尾の特徴
- 良い例/悪い例

## 発信テーマ
- テーマ1
- テーマ2

## 禁止事項
- ○○は使わない
```

## 外部クライアントの追加（オプション）

`~/clients/{クライアント名}/` に以下の構成でフォルダを作成すると、アプリに自動表示されます:

```
共有/
├── ガイド/
│   ├── ブランドボイス.md
│   └── ターゲット.md
├── テンプレート/
│   └── x-post.md
└── 品質基準/
    └── 採点ルブリック.md
```
