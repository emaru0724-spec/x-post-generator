# コンテンツ採点ルブリック（全制作物共通）

## 使い方
- 複数案を生成する場合、**全案を採点してランク付け**してから出力する
- 1案のみの場合も採点し、B以下なら自動リライトしてから出力する

---

## 採点軸（各10点満点 → 合計50点）

### 1. フック力（冒頭の引き込み）
| 点数 | 基準 |
|------|------|
| 9-10 | スクロールを止める。「え、何これ」と思わせる。意外性がある |
| 7-8 | 興味を引く。読み進めたくなる |
| 5-6 | 普通。目に留まるが引き込みが弱い |
| 3-4 | 平凡。よくある書き出しで流される |
| 1-2 | 退屈。AI感が強い。テンプレ丸出し |

### 2. ターゲット共感度
| 点数 | 基準 |
|------|------|
| 9-10 | 「完全に俺のことじゃん」と声に出すレベル。具体的な状況描写が刺さる |
| 7-8 | 「あ、自分にも当てはまる」と感じる |
| 5-6 | ターゲットに向いてはいるが、ボヤけている |
| 3-4 | 誰に言っているか不明確 |
| 1-2 | ターゲット不在。万人向けの空虚なコピー |

### 3. ブランドボイス一致度
| 点数 | 基準 |
|------|------|
| 9-10 | そのクライアント本人が書いたとしか思えない口調。禁止ワードゼロ |
| 7-8 | 口調は合っている。わずかに違和感がある程度 |
| 5-6 | 方向性は合っているが、所々で「この人こんな言い方しない」がある |
| 3-4 | AI感が出ている。定型表現が混在 |
| 1-2 | 完全にAIが書いた文章。brand-voiceを無視 |

### 4. 具体性（数字・体験・証拠）
| 点数 | 基準 |
|------|------|
| 9-10 | 数字+実体験+固有名詞がセットで入っている。説得力が段違い |
| 7-8 | 数字と具体例が1つ以上。十分な説得力 |
| 5-6 | 数字はあるが、ありきたり。体験談がない |
| 3-4 | 抽象的な表現が多い。「すごい」「簡単」等の形容詞頼み |
| 1-2 | 全て抽象的。根拠なし |

### 5. CTA効果（行動喚起）
| 点数 | 基準 |
|------|------|
| 9-10 | 読んだ直後に行動したくなる。CTA前の理由付けが完璧 |
| 7-8 | 自然にCTAに繋がる。違和感なし |
| 5-6 | CTAはあるが唐突。流れが不自然 |
| 3-4 | CTAが弱い or 押し売り感がある |
| 1-2 | CTAなし、または的外れ |

---

## ランク判定

| ランク | 合計点 | 判定 |
|--------|--------|------|
| **S** | 45-50 | 即採用。手直し不要 |
| **A** | 40-44 | 採用候補。微調整で使える |
| **B** | 35-39 | 使えるが、弱い項目のリライト推奨 |
| **C** | 30-34 | リライト必須。方向性は合っている |
| **D** | 29以下 | 作り直し。根本的に見直す |

## 制作物別の加点要素

### X投稿
- **バズ加点**: RT/引用RTしたくなる構造（+1〜+3）
- **保存加点**: 後で見返したい有益情報（+1〜+2）

### ステップ配信
- **開封期待加点**: 次回を読みたくなる仕掛け（+1〜+3）
- **教育効果加点**: 読後の理解度変化（+1〜+2）

### LP・VSL台本
- **転換加点**: 「申し込まない理由がない」状態に持ち込めているか（+1〜+5）

### note記事
- **スキ加点**: シェアしたくなる共感ポイント（+1〜+3）
- **CTA加点**: LINE登録等への自然な導線（+1〜+2）

### YouTube台本
- **視聴維持加点**: 「ここで離脱しない」仕掛けの数（+1〜+3）
- **登録加点**: チャンネル登録したくなるCTA（+1〜+2）

加点はランク判定には含めず、参考指標として表示する。
